## 🎮 How to Play?

1. **Select Level & Topic** from dropdowns 📚
2. **Read the Question** on flashcard 🤔
3. **Tap to Flip** and see the answer 🔄
4. **Mark** ✅ if you got it, ❌ if not
5. **Complete Topic** → Take the **Quiz** 📝
6. **Score 80%+** → You're a Topic Master! 👑
